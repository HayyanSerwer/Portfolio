## Portfolio Website

A personal portfolio built with React, Three.js, and a lot of scroll-driven animation. Features a retro CRT monitor as the hero, an interactive projects section with 3D floppy disks, a vertical career timeline, and a contact section, all as one seamless scroll experience.

You can also click on the monitor after the screen becomes visible to display an interactive windows-XP themed experience.


### Tech Stack

React + TypeScript
Three.js / React Three Fiber / Drei
Tailwind CSS
Vite
